# cococonnections

This project is my recreation of a popular word puzzle game. The goal of the game is to find groups of four words that share a common theme or connection. 
Players are shown a grid of words and must identify four groups of related words. When a correct group is selected, it is revealed and removed from the board.

When the game loads, the JavaScript reads the puzzle data from the JSON file, displays the words on the board, and checks whether the player's selections match the correct groups.

I created this project to practice building interactive browser games and to learn how to separate game logic, page layout and data storage using basic web technologies. It also offers a more personal version of this game for my friends to enjoy.
